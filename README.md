<h1 align="right">🛒 Amazon Shopping Dashboard — Data Analysis Project 📊</h1>

<p>
An interactive data analysis project built using Power BI to visualize and analyze Amazon sales data.
This project focuses on transforming raw data into meaningful insights related to sales performance,
customer behavior, and product trends.
</p>

<h2>📌 Project Overview</h2>
<p>
The Amazon Shopping Dashboard provides a comprehensive view of business performance by analyzing
sales, revenue, and customer data. The dashboard helps in identifying trends, top-performing products,
and regional performance through interactive visualizations.
</p>

<h2>🧠 Theory / Concept</h2>
<p>
This project is based on the principles of <b>Business Intelligence (BI)</b> and <b>Data Visualization</b>.
Raw data collected from Excel is cleaned and transformed before being visualized in Power BI.
Dashboards allow users to interpret complex datasets easily through charts, graphs, and KPIs.
</p>

<p>
By using interactive filters and visuals, users can explore different aspects of the data such as
region-wise performance, product category trends, and time-based sales analysis. This approach helps
businesses make data-driven decisions effectively.
</p>
