🛒 Amazon Shopping App Dashboard 📊

An interactive and visually appealing Amazon Shopping Dashboard built to analyze sales performance, customer behavior, and product trends.

✨ Key Features:

📈 Sales performance overview (daily, monthly, yearly)
🧑‍🤝‍🧑 Customer insights & segmentation
🛍️ Product category analysis
🌍 Region-wise sales distribution
💰 Revenue & profit tracking
🔍 Dynamic filters for deep data exploration

🧰 Tools & Technologies Used:

📊 Power BI
📂 Excel (Data Source)
🧹 Data Cleaning & Transformation
📉 Data Visualization Techniques

🚀 Project Highlights:

Designed a user-friendly and interactive dashboard
Converted raw data into meaningful insights
Helped in identifying top-performing products and regions
Improved decision-making through visual storytelling

📌 Use Case:
This dashboard can be used by businesses to track sales performance, understand customer trends, and optimize product strategies.
